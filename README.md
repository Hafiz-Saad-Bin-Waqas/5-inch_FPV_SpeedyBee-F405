# 5-inch FPV Drone Build (Betaflight F405)

A clean and simple 5-inch FPV build focused on reliability, smooth flight, and proper documentation.

## Parts List
- **Frame:** [Mark4 5-inch](https://thriftyrcstore.com/product/mark-4-5inch-225mm-carbon-fiber-frame-for-fpv-freestyle-rc-racing-drone/)
- **Motors:** [T-Motor Velox V2 2306.5 1950KV  ](https://www.smarthobby.pk/product-page/v2306-5-v2-1950-kv-royal-blue)
- **FC + ESC:** SpeedyBee F405 Stack (BLHeli_32)  
- **Camera:** [Caddx Rattel 2](https://www.smarthobby.pk/product-page/caddx-ratel-camera-2)
- **Receiver:** [FlySky FS-iA6B](https://www.smarthobby.pk/product-page/copy-of-flysky-fs-ia6b-2-4g-6ch-ppm-amp-ibus-receiver)
- **Battery:** 4S 1500mAh 100C CNHL
- **Props:** [Gemfan Freestyle3S 5.1x3x3 Durable F3 Prop
](https://www.smarthobby.pk/product-page/gemfan-freestyle3s-5-1x3x3-durable-f3-prop)
- **VTX:** [Ultra FPV 1600W vtx
](https://www.smarthobby.pk/product-page/ultra-fpv-1600w-vtx)
- **Antenna for vtx:** [5.8Ghz Lollipop3 3dBi RHCP Antenna SMA](https://www.smarthobby.pk/product-page/5-8ghz-lollipop3-3dbi-rhcp-antenna)
- **Accessories:** XT60, heatshrink, straps, zip ties  

**Estimated Total:** $347.43

## Build Summary
- Fully soldered and assembled from scratch  
- Betaflight configured (modes, rates, OSD, RPM filters)  
- Motors calibrated + BLHeli_32 settings tuned  
- Receiver bound and failsafe verified
