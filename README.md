# 5-inch FPV Drone Build (Betaflight F405)

A clean and simple 5-inch FPV build focused on reliability, smooth flight, and proper documentation.

## Parts List
- **Frame:** Mark5 5-inch  
- **Motors:** T-Motor Velox V2 2306.5 1950KV  
- **FC + ESC:** SpeedyBee F405 Stack (BLHeli_32)  
- **Camera:** Caddx Ant  
- **Receiver:** FlySky FS-iA10B  
- **Battery:** 4S 1500mAh 100C CNHL  3x
- **Props:** Gemfan 51466 / HQ 5x4.3x3
- **VTX:** SpeedyBee TX800 + Foxeer Lollipop 5.8GHz RHCP Antenna
- **Accessories:** XT60, heatshrink, straps, zip ties  

**Estimated Total:** $275 - $420
(The big range is because this project highly depends on where i buy these things from and the shipping costs, taxes, etc)

## Build Summary
- Fully soldered and assembled from scratch  
- Betaflight configured (modes, rates, OSD, RPM filters)  
- Motors calibrated + BLHeli_32 settings tuned  
- Receiver bound and failsafe verified  

## Performance Notes
- Smooth, responsive, good for freestyle  
- ~3:30–5:00 flight time on 4S 1500mAh  
- Stable hover, minimal vibration  

## Notes
This repo is mostly for documenting progress and storing media, diagrams, and Betaflight settings.
